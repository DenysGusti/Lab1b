Interactive Controls

1. Selecting Objects

  • Press 1-9 → Select a shape.
  • Press 0 → Select all shapes.
  • Press Space → Toggle camera mode.
  • Press L → Toggle light mode.

2. Scaling Objects

  • a: Decrease width | A: Increase width
  • b: Decrease height | B: Increase height
  • c: Decrease depth | C: Increase depth

3. Rotating Objects / Light

  • i/k: Rotate around X-axis
  • o/u: Rotate around Y-axis
  • l/j: Rotate around Z-axis

4. Moving Objects / Light / Camera

  • →: Move right | ←: Move left
  • ↑: Move up | ↓: Move down
  • ,: Move forward | .: Move backward

5. Shading and Illumination Modes

  • w: Gouraud Shading with Diffuse Lighting
  • e: Gouraud Shading with Specular Highlights
  • r: Phong Shading with Diffuse Lighting
  • t: Phong Shading with Specular Highlights
